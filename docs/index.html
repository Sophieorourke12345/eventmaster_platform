Booking policies.

These policies explain how EventSpace handles booking support, cancellations and refund requests for ticket purchases made through the platform.

Refunds

Refund eligibility is set by the organiser and displayed with the event information. Guests can contact EventSpace support if a refund agreed under the stated policy has not been received.

Cancellations

If an event is cancelled, ticket holders will be notified using their booking email and refunded to the original payment method. Processing times can vary by bank.

Questions or disputes

For booking, event or payment support, email sophorourke2000@gmail.com. Please include the event name and booking reference.
